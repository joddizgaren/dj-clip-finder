Place your app icon here before building:

  icon.ico  — Windows installer icon (256x256 recommended)

How to create one (free):
  1. Open any square image (your logo, a DJ icon, etc.)
  2. Go to https://icoconvert.com
  3. Upload the image, select "256x256", click Convert
  4. Download and save as icon.ico in this folder

Without icon.ico the build still works but uses a default Electron icon.
